# JavaScript 基础补充

## 对象

### JS 中的数据类型

JS 中的数据类型分为两大类：**原始值**和**对象**，或者称为**基本数据类型**和**引用数据类型**，其中，

原始值包括：

1. 数值 Number
2. 大整数 BigInt
3. 字符串 String
4. 布尔值 Boolean
5. 空值 Null
6. 未定义 Undefined
7. 符号 Symbol

对象包括：

1. 对象 Object
2. 函数 Function
3. 数组 Array

### 什么是对象？

对象是 JS 中的一种复合数据类型，它相当于一个容器，在对象中可以存储不同类型的数据。
